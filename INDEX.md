## 🔑 Key Concepts

### Game Mechanics
- **Keys**: Currency earned from tasks (1 key = 1 spin)
- **Wheel**: Game mechanic where keys are spent for prizes
- **Balance**: Coins earned from wheel spins (sent to wallet)
- **Wallet**: TON wallet for withdrawing balance
- **Tasks**: Ad-watching system to earn keys

### Technical
- **Frontend**: React + Next.js + TypeScript
- **State**: React Context (AppContext)
- **API**: RESTful with JWT auth
- **Database**: SQL-based (PostgreSQL/MySQL)
- **Blockchain**: TON for withdrawals

### Security
- Telegram verification
- JWT authentication
- Authorization headers
- Input validation
- Transaction logging
