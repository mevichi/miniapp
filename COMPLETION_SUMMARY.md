## 🎮 Game Features

### Gameplay Flow
1. **Earn Keys**: Complete tasks, watch 5-second ads
2. **Spin Wheel**: Use keys to play (1 key = 1 spin)
3. **Win Coins**: Get random prizes (0-100 coins)
4. **Withdraw**: Connect TON wallet and cash out

### Reward System
- Task completion: +1 Key
- Wheel spinning: Costs 1 Key
- Prizes: 5, 10, 20, 50, 100 coins (or nothing)

### Balance Tracking
- Real-time updates
- Backend verification
- Transaction history
- Withdrawal support
